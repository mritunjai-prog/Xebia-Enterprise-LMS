## 📖 About the Project

**Xebia Enterprise LMS** is a state-of-the-art, high-performance educational platform designed to streamline corporate training and academic learning. Engineered with a scalable microservices backend and a highly interactive, modern React frontend, it empowers educators to seamlessly build, manage, and deliver highly structured course curriculums. 

The platform supports a deep content hierarchy (**Courses** ➔ **Modules** ➔ **Submodules** ➔ **Content Blocks**) to organize learning materials intuitively while providing comprehensive analytics and student tracking features.

---

## ✨ Key Features

- 🎯 **Advanced Curriculum Builder:** An intuitive, fully animated drag-and-drop interface for structuring complex courses, complete with real-time status updates and order mapping.
- ⚡ **Comprehensive Analytics:** Built-in dashboard to monitor key performance indicators (KPIs), batch progress, and overall student engagement.
- 🔍 **SEO & Metadata Engineering:** Deep SEO configuration at the submodule level, featuring canonical URLs, Open Graph integrations, and dynamic slug generation.
- 🎨 **Premium Aesthetics:** Stunning UI/UX featuring glassmorphism elements, dynamic micro-animations (Framer Motion), and seamless light/dark mode transitions.
- ⚙️ **Offline First / Local Sync:** Fallback data persistence capabilities using local browser storage during development phases, preparing for strict data synchronization with the backend.

---

## 🛠️ Technology Stack

**Frontend Ecosystem:**
- **Core Framework:** [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Routing:** [Tanstack Router](https://tanstack.com/router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) + Custom CSS Variables
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **State Management:** [Zustand](https://github.com/pmndrs/zustand)
- **Icons:** [Lucide React](https://lucide.dev/)

**Backend Architecture (Java Integration):**
- **Core:** Spring Boot & Java 17+
- **Database:** JPA / Hibernate (MySQL/PostgreSQL)
- **Architecture:** Microservices with Service Discovery and API Gateways.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites
Make sure you have Node.js and npm installed on your machine.
- Node.js (v18 or higher)
- npm (v9 or higher)

### Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/mritunjai-prog/Xebia-Enterprise-LMS.git
   ```
2. Navigate into the project directory:
   ```sh
   cd Xebia-Enterprise-LMS
   ```
3. Install NPM packages:
   ```sh
   npm install
   ```
4. Start the Vite development server:
   ```sh
   npm run dev
   ```
5. Open your browser and navigate to `http://localhost:8080/`.
