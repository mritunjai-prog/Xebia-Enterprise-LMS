# Mobile Responsiveness Plan — Full Frontend

## Scope
Make the entire Xebia Enterprise LMS frontend fully responsive across mobile (320-480px), tablet (481-768px), laptop (769-1024px), and desktop (1025px+).

## Guiding Principles
- **Mobile-first**: Start with mobile layout, scale up with `sm:`, `md:`, `lg:`, `xl:` breakpoints
- **WCAG 2.5.5**: All touch targets ≥ 44×44px
- **Minimum readable text**: 12px on mobile (no text below `text-xs`)
- **No horizontal overflow** on any viewport
- **Preserve desktop experience** — changes should only add responsive behavior, not alter desktop layout
