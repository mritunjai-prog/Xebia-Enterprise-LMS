# UI Overhaul Plan — Assessment Builder & Batch Management

## Problems Identified

### Assessment Builder (`src/pages/AssessmentBuilder.jsx`)
1. **Table view alignment**: Actions header has extra `pr-6` mismatch vs data cells
2. **Grid view action buttons**: Edit button uses `flex-1` and dominates on non-draft assessments (Duplicate/Delete are tiny squares next to a wide Edit bar)
3. **Grid card header**: Uses `items-start` — status badge doesn't center with title
4. **Marks/Duration column**: Two-line layout (marks on top, duration below) with `text-right` — visually cramped
5. **Table has `rounded-2xl` on wrapper**: User wants sharp table edges (already fixed)

### Batch Management (`src/pages/BatchManagement.jsx`)
1. **Course name not visible**: `editBatch()` passes `""` as course (line 251), wiping the course name on every edit
2. **Sort bug**: "Newest" and "Oldest" use identical comparator (both `b - a`)
3. **Grid card context menu clips**: `overflow-hidden` on card clips the `bottom-full` dropdown
4. **Grid card action buttons**: Hidden behind a 3-dot menu instead of being directly visible like assessment builder

---

## Changes

### 1. Assessment Builder — Table View Alignment Fix
**File**: `src/pages/AssessmentBuilder.jsx`

- Fix `<th>` Actions header: remove extra `pr-6`, use consistent `px-6 text-right`
- Ensure all `<td>` cells in each column use consistent padding with their `<th>`
- Make Marks/Duration a single-line layout: "10 pts · 10 min" instead of stacked

### 2. Assessment Builder — Grid View Card Redesign
**File**: `src/pages/AssessmentBuilder.jsx`

- Change card header from `items-start` to `items-center` so status badge centers with title
- Make all 4 action buttons icon-only (same size) in a `flex items-center gap-1.5` row — no `flex-1` on any button, consistent visual weight
- Add `space-y-3` between info chips and actions for breathing room

### 3. Batch Management — Course Name Fix
**File**: `src/pages/BatchManagement.jsx`

- In `handleEditSubmit` (line ~248): Look up the course name from `adminCourses` using the batch's `courseId`, or preserve the existing `batch.course` value instead of passing `""`
- Add a Course dropdown to the edit modal (same as create modal)

### 4. Batch Management — Sort Fix
**File**: `src/pages/BatchManagement.jsx`

- Fix "Oldest" comparator: `new Date(a.createdAt) - new Date(b.createdAt)`

### 5. Batch Management — Grid Card Menu Fix
**File**: `src/pages/BatchManagement.jsx`

- Move the context menu dropdown outside the card (use a portal or change overflow to `overflow-visible` on the card with `overflow-hidden` only on the cover image area)
